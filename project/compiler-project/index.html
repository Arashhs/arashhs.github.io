Compiler Project

Aug 19, 2020
Code
© Arash Hajisafi
  • Lexical Analysis, Syntax Analysis, Semantic Analysis, and Intermediate Code Generation implemented using JFlex and Bison for my Compiler Design course project for the grammar which is included in the repository
  • Generated code representation is in three address code format

How to compile

Make sure that you have set up Bison and Flex properly, then run the following code in terminal:

bison -L JAVA SHLang.yac -v
del SHLang.java
ren SHLang.cac SHLang.java
javac Yylex.java SHLang.java
java YYParser
gcc compiler.c

Alternatively, you can run the run.bat file.

Example

Input:

Program p1Main;
	Real n1, n2, n3, n4, n5;
Begin
	
	n1 := #20 - #10 / #2;
	n2 := n1 / (#3 + #2);

	If(n1 .GT. #2) Then Begin
	
		For n4:=n2 Downto #0 Do Begin
			n3 := n3 + #1;
			n2 := n2 - #1
		End
	End Else
		If(n2 .NE. #3) Then n2 := #80


End;
